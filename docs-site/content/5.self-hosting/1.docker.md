## Quick Start

```bash
docker run -d --name zvault \
  -p 8200:8200 \
  -v zvault-data:/data \
  ghcr.io/nicosalm/zvault:latest
```

## Docker Compose

```yaml
version: '3.8'
services:
  zvault:
    image: ghcr.io/nicosalm/zvault:latest
    ports:
      - "8200:8200"
    volumes:
      - zvault-data:/data
    environment:
      - ZVAULT_LISTEN_ADDR=0.0.0.0:8200
      - ZVAULT_STORAGE_PATH=/data
      - RUST_LOG=info
    restart: unless-stopped

volumes:
  zvault-data:
```


## Environment Variables

| Variable | Default | Description |
|----------|---------|-------------|
| `ZVAULT_LISTEN_ADDR` | `127.0.0.1:8200` | Listen address (use `0.0.0.0:8200` in Docker) |
| `ZVAULT_STORAGE_PATH` | `./data` | Path to persistent storage |
| `RUST_LOG` | `info` | Log level (`debug`, `info`, `warn`, `error`) |

## Persistent Storage

Always mount a volume for `/data` to persist your encrypted secrets across container restarts. Without a volume, all data is lost when the container stops.
