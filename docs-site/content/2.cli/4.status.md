## Status Fields

| Field | Description |
|-------|-------------|
| Initialized | Whether the vault has been initialized with `zvault init` |
| Seal Status | `SEALED` or `UNSEALED` |
| Total Shares | Number of Shamir key shares generated during init |
| Threshold | Minimum shares required to unseal |
| Unseal Progress | How many shares have been submitted (only shown when sealed) |
